## 使用

### 克隆

克隆(clone)到你的本机，运行命令：

```
npm run install
or 淘宝镜像
cnpm run install
```

> 若进行二次开发，则最好
>
> 修改 `webpack.config.js`文件里的`mode`属性的值为`development`（开发模式）
>
> 默认为`production`(生产环境)

### 打包

运行命令

```
npm run build
```

进行打包，打包在`dist`目录

### 数据

数据存在`data.json`文件
